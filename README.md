# Auto Proxy Fetcher

Automatischer Proxy-Server Fetcher mit GitHub Actions.

## Features

- Automatisches Fetchen von Proxy-Servern alle 6 Stunden
- Validierung der Proxies
- Automatisches Update der `proxies.txt`
- GitHub Actions Workflow
